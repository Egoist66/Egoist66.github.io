1. Данное изображения можно реализовать через написание svg объектов вложенных друг в друга
  2. Данное изображения можно реализовать через создание svg объектов используя соответствующий софт проектирующий svg формы (наиболее удобный способ)
  3. Данное изображения можно так же реализовать через использование соответсвующих CSS свойств касаемо границ блочных объектов и их манипуляции
  4. Можно так же использовать графический редактор (photoshop) или аналоги создав данные объекты вкладывая один слой в другой PSDETCH Photopea
